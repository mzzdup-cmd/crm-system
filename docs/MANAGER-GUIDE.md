# CRM School — памятка менеджера

**Сайт:** https://crm-school-v2.web.app  
**Логин:** ваш рабочий аккаунт (выдаёт руководитель)

---

## Главное правило перехода

| Когда | Куда вносим оплаты |
|---|---|
| **Июнь и раньше** | Google Таблицы (как раньше) |
| **С 1 июля — все новые продажи** | **CRM → Новая оплата** |
| **Июньские подписчики в CRM** | Заносим **в июле** (инструкция ниже) |

**Не дублируйте одну и ту же оплату** и в таблице, и в CRM. Июльская оплата — только CRM.

---

## Меню — что где лежит

| Раздел | Зачем заходить |
|---|---|
| **Главная** | Задачи на сегодня: оплаты, просрочки, VK, KPI месяца, отпуск |
| **Продажи** | Ваши оплаты, клиенты, сделки, подписки |
| **Новая оплата** | Внести любую новую оплату |
| **Быстрые продажи** | Дописать черновик, начатый на ходу |
| **Уведомления** | Напоминания: просрочки, VK, график |
| **Зарплата** | Ваш расчёт за **текущий месяц** + архив прошлого |
| **Календарь** | Кто в отпуске / выходной |
| **Запросы** | Подать выходной или отпуск |
| **База знаний** | Полезные ссылки |

**Вам недоступны:** MM Аналитика, Управление (это у руководителя).

## Новая оплата — 3 типа ситуаций

### 1. Новый клиент (с 1 июля)

1. **Новая оплата**
2. Сделка: **Новая** / **ББ** / **Рассылка**
3. Вставить **ссылку на диалог** Bluesales
4. Заполнить: сумма, дата оплаты, поток, VK, откуда (traffic), платёжная система
5. Сохранить

---

### 2. Доплата — клиент уже есть в CRM

1. Сделка: **Доплата …** (по типу: Новая, ББ, Апсэйл и т.д.)
2. Ссылка на диалог → CRM **находит клиента**
3. Сумма + дата оплаты → Сохранить

---

### 3. Клиент из Google ТТ — «Клиент из таблицы (без карточки CRM)»

Клиент жил только в Google ТТ — **карточки в CRM нет**, поэтому «Доплата Новая» его не находит (ссылка на диалог **одна и та же**).

1. Режим: **Клиент из таблицы (без карточки CRM)**
2. Ссылка на диалог (или ID из БС) → **Найти**
3. **Тип сделки для ТТ** — выбрать реальный тип для колонки B:
   - после «Новой» в июне → **Доплата Новая**
   - после «ББ» → **Доплата ББ**
   - новый курс → **Апсэйл** / **Доплата Апсэйл**
4. **Если нашли** — сумма, дата, платёжка, номер счёта → **Сохранить доплату**
5. **Если не нашли (первый раз)** — данные + **бюджет (сумма тарифа)** + оплата → **Сохранить**

> Карточка клиента в CRM **не создаётся**. Повторные доплаты — **«Найти»** по **той же ссылке**.

---

### 4. Доплата клиента, который уже в CRM

1. Сделка: **Доплата …** (по типу)
2. Ссылка на диалог → CRM **находит клиента**
3. Сумма + дата оплаты → Сохранить

Если клиент из таблицы (июнь) — не используйте этот путь, см. раздел 3.

## Типы сделок (шпаргалка)

| Сделка | Когда |
|---|---|
| **Новая**, **ББ**, **Рассылка** | Новый клиент |
| **Доплата …** | Клиент уже в CRM |
| **Апсэйл** / **Доплата Апсэйл** | Апсейл |
| **Возврат** | Возврат денег |
| **Доплата (старый подписчик)** | Устарело → используйте **Июньский подписчик (из таблицы)** |

## Зарплата

- Считается **по календарному месяцу**
- **Июль** = только оплаты, смены и бонусы **июля**
- На главной — сумма ЗП **текущего** месяца
- Внизу страницы «Зарплата» — **Архив** за прошлый месяц

Состав: оклад 50 000 ₽ + 5% от продаж + бонус оборота + ночные + ручные бонусы.

---

## Главная — что смотреть каждый день

- **Оплата сегодня / завтра**
- **Просрочено**
- **Не заполнен VK** — дозаполнить в карточке клиента
- **Черновики продаж** — дописать или удалить
- **Мои KPI** — ваша выручка за месяц

---

## Календарь и запросы

- **Запросы** → выходной (один день) или отпуск (диапазон дат)
- Статус: *На рассмотрении* → *Одобрено* / *Отклонено*
- Одобренное видно в **Календаре** у всей команды

---

## Выгрузка в Google ТТ

Оплаты из CRM **автоматически** добавляются в вашу ТТ-таблицу **3 раза в день** (01:00, 13:00, 20:00 МСК).

Проверяйте, что новая оплата появилась в таблице на следующий слот выгрузки.

## Чек-лист — первая неделя июля

- [ ] Зайти в CRM, посмотреть **Главную**
- [ ] Занести **июньских подписчиков** (раздел 3 выше)
- [ ] Все **новые** оплаты — только **Новая оплата** в CRM
- [ ] Дозаполнить **VK**, где CRM просит
- [ ] Проверить **Зарплату** за июль
- [ ] Подать **Запросы**, если нужен отпуск/выходной
