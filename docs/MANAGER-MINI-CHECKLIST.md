# CRM — мини-чеклист менеджера

**Сайт:** https://crm-school-v2.web.app  
**Когда:** первые дни после 1 июля, потом — только по пунктам «каждый день».

---

## День 1 — один раз пройти все вкладки

Везде вверху/слева должно быть **ваше имя** и роль **Менеджер**. Если имени нет — напишите руководителю.

| # | Вкладка | Что проверить |
|---|---------|---------------|
| 1 | **Главная** | Видны блоки: KPI месяца, задачи, просрочки (не пустой экран) |
| 2 | **Продажи → Оплаты** | Есть ваши записи или пусто — но страница открывается |
| 3 | **Продажи → Клиенты** | Только ваши клиенты |
| 4 | **Продажи → Подписки** | Список подписок загружается |
| 5 | **Новая оплата** | Форма открывается, менеджер = вы |
| 6 | **Быстрые продажи** | Кнопка «Быстрая продажа» есть; блок «Ожидают подтверждения» |
| 7 | **Уведомления** | Страница открывается (может быть пусто) |
| 8 | **Зарплата** | Загружается без ошибки; месяц = **текущий** (июль) |
| 9 | **Календарь** | Открывается, видны отпуска/выходные команды |
| 10 | **Запросы** | Можно подать выходной / отпуск |
| 11 | **База знаний** | Открывается |

**MM Аналитика** и **Управление** — вам не видны, это нормально.

---

## Каждый день (2–3 минуты)

- [ ] **Главная** — оплаты сегодня/завтра, просрочки, «не заполнен VK»
- [ ] **Уведомления** (колокольчик) — нет ли срочного
- [ ] Новые оплаты — только через **Новая оплата** (не в Google Таблицу)

---

## После каждой новой оплаты — сразу (1 минута)

- [ ] **Главная** — выручка месяца / KPI обновились
- [ ] **Продажи → Оплаты** — запись на месте, сумма и дата верные
- [ ] **Продажи → Клиенты** — клиент появился или обновился (кроме клиента из таблицы — см. ниже)
- [ ] Если CRM просит **VK** — дозаполнить в карточке клиента

> Обновление на экране — **сразу**. В Google ТТ — **не сразу** (см. следующий блок).

---

## Google ТТ — проверять только **первые 3 дня** июля

Выгрузка из CRM в вашу ТТ-таблицу: **3 раза в день** — 01:00, 13:00, 20:00 **МСК**.

| Когда внесли оплату в CRM | Когда смотреть ТТ |
|---------------------------|-------------------|
| До ~13:00 | После 13:00 того же дня или утром след. |
| До ~20:00 | После 20:00 или утром след. |
| После 20:00 | Утром **следующего** дня (после 01:00) |

**Чеклист (дни 1–3 июля):**
- [ ] День 1: тестовая/реальная оплата → на след. слот проверили строку в **ТТ**
- [ ] День 2: ещё одна оплата → снова проверили ТТ
- [ ] День 3: последняя проверка ТТ

**С 4 июля:** если все 3 дня выгрузка была ок — **ТТ больше не проверяем**, работаем только в CRM.

Если строки нет 2 дня подряд — скрин + ссылка на диалог → руководителю.

## Клиенты из Google ТТ (июнь) — одна схема

**Почему не через «Доплата Новая»?**  
CRM ищет клиента в **карточках**. Июньские клиенты живут только в Google ТТ — **карточки в CRM нет**. Ссылка на диалог **одна и та же**.

### Всегда один путь

1. **Новая оплата**
2. Режим: **«Клиент из таблицы (без карточки CRM)»**
3. Ссылка на диалог (или ID из БС) → **«Найти»**
4. **Тип сделки для ТТ** — выбрать реальный тип (как в Google ТТ):
   - доплата после «Новой» в июне → **Доплата Новая**
   - доплата после «ББ» → **Доплата ББ**
   - доплата после «Рассылки» → **Доплата Рассылка**
   - новый курс у старого клиента → **Апсэйл** / **Доплата Апсэйл**

**Если нашли** (уже вносили):
- CRM подставит имя, курс, тариф, остаток
- Меняете **тип для ТТ** (если нужно), **сумму, дату, платёжку, номер счёта**

**Если не нашли** (первый раз):
- Заполнить данные + **бюджет (сумма тарифа)** + оплату один раз
- Следующие доплаты — снова **«Найти»** по **той же ссылке**

Карточки клиента в CRM **не будет** — только записи оплат. Это нормально.

---

## Главные правила (не забыть)

| ✅ Так | ❌ Не так |
|--------|----------|
| Июльские **новые** клиенты → «Новая» / «ББ» / «Рассылка» | Июль в Google Таблицу |
| Клиент из таблицы → режим «без карточки» + **тип для ТТ** + **та же ссылка** | Искать через «Доплата Новая» (карточки нет) |
| В Google ТТ колонка B = выбранный тип («Доплата ББ» и т.д.) | Ждать, что ТТ сама поймёт смысл |

---

## Если что-то не так

1. **Ctrl+F5** (обновить страницу)
2. Проверить, что вошли под **своим** email
3. Руководителю: что делали + скрин + ссылка на диалог
